JSON

JavaScript Object Notation

Def: The JSON object contains methods for parsing JavaScript Object Notation (JSON) and converting values to JSON.

Lightweight data-interchange Format

    DATA TYPES:

  • NUMBERS: No difference between integer and floats
  • STRING: String of Unicode characters. Uses double quotes
  • Boolean: True or False
  • Array: Ordered list of 0 or more values
  • OBJECT: Unordered collection of key/value pairs
  • Null:Empty value

    JSON Syntax Rules:

  • Uses key/value pairs -{"name":"Norbert"}
  • Uses double quotes around KEY and Value
  • Must use the specified data type
  • File type is .json
  • MIME type is "Application/json"
